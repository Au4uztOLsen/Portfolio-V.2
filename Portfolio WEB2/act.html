COMPUTER ENGINEERING AND DATA SCIENCE

LINE Chat Bot by Dialogflow

สร้างเเชตบอตไลน์ที่เป็น Messaging API ใน Line Developers เเล้วนำมาเชื่อมกับ Dialogflow เพื่อรับเเละตอบกลับข้อความที่ได้สร้างไว้ใน Intent ของ Dialogflow โดยในบาง Intent ก็จะมีการนำโค้ดจาก LineBot Designer ที่เป็นประเภท JSON เพื่อนำมาสร้าง Flex Message ที่สะดวกต่อ User ในการถามตอบกับเเชทบอท
